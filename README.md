## A Vue model

### 项目结构

```
├── bankend                   ........... 后端目录
├── mock
|   └── api                   ........... mock数据
├── public                    ........... 公共资源目录
├── source                    ........... vue源码分析目录
|   ├── router                ........... vue-router 路由源码分析目录
|   └── vue.js                ........... vue2.5.16 源码分析文件
├── src
│   ├── assets                ........... 项目内置资源目录
│   ├── components            ........... 公共组件目录
|   ├── configs               ........... 配置文件目录
│   ├── plugins               ........... vue插件目录
│   ├── router                ........... vue路由配置目录
│   ├── store                 ........... vuex配置目录
|   ├── utils                 ........... 实用工具目录
│   ├── views                 ........... 页面视图文件
|   └── ...                   ........... 待续
├── .browserslistrc           ........... 浏览器兼容性描述
├── .prettierrc               ........... prettier配置
├── .babel.config.js          ........... babel配置
├── bankend.config.js         ........... 后端基础配置
├── bankend.devServer.js      ........... 后端服务配置
├── vue.config.devServer.js   ........... vue代理服务配置
└── vue.config.js             ........... vue基础配置
```
